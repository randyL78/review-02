Detailed Description

A standard C++ STL style iterator.

Recall the rules on Class naming and the STL.

Note I have relaxed my rule regarding definitions within class declarations.

These are rudimentary iterators. There are a number of additions needed before we can declare this complete–e.g., operator-> and iterator traits. The latter is beyond the scope of this course.
